Para executar o projeto, basta ter o R instalado e abrir o diretório da pasta default_credit_card_clients como diretório de trabalho. O arquivo de dados default_credit_card_clients.xls deve estar na subpasta data/, e os scripts auxiliares na pasta scripts/ exatamente como entregues. Com o diretório configurado, é suficiente rodar no console: source("main.R", encoding = "UTF-8"). O script principal carrega automaticamente os pacotes necessários (instalando-os se não existirem), faz o pré-processamento dos dados, a análise exploratória, o PCA, a seleção de variáveis por algoritmo genético, o ajuste da regressão logística, a validação cruzada e gera os gráficos, salvando as figuras na pasta figs/ e imprimindo as métricas e interpretações no console.
